# 行列のパズル

Aをm×nの実数の行列とする. Aについて好きな行または好きな列にある全ての数字の符号を一斉に変更する操作を考える.
この操作を繰り返すことで, どの行の和やどの列の和がすべて負でない行列にできることを証明せよ.
