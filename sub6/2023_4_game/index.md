# 石取りゲーム

AさんとBさんが次のゲームをします.

- 5個のコインの山があって, それぞれの山には5,8,6,10,7枚のコインがある.
- AさんとBさんが交互に"一つの山"を選び, そこからコインを取ります. ただし一度にとるコインの枚数は1,3,4枚のいずれかと等しくないといけない.
- 最後にコインを取った方が勝ち.
- 両者とも最善を尽くして戦う. 

最初の手番がAさんであるとき, どちらが勝つでしょうか?
